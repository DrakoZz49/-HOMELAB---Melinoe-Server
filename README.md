De día escribo código para que las empresas funcionen mejor.

## ¿Qué hago aquí?

Durante el día trabajo desarrollando soluciones de software. Cuando cierro el portátil del trabajo, abro el otro portátil. El que zumba en un rincón de casa sirviendo películas, libros y copias de seguridad.

No tengo un rack. No tengo un CPD. Tengo un portátil reciclado, mucha curiosidad y demasiadas ganas de cacharrear.

---

## 🖥️ El proyecto estrella

### [Melinoé Homelab](https://github.com/byDrakoZz/melinoe-homelab)

Mi servidor doméstico. Un portátil reconvertido en servidor corriendo Ubuntu Server con Docker.  
Aquí documento todo: servicios, configuraciones, y los errores que me costaron horas para que a ti te cuesten minutos.

**Stack actual:**
- 🎬 Jellyfin — streaming de media
- ☁️ Nextcloud — mi nube personal
- 📚 Calibre-Web — biblioteca digital
- 🎮 RomM — gestión de ROMs
- 🔥 Flame — dashboard de servicios
- 📡 Uptime Kuma — monitorización
- 🐳 Watchtower — actualizaciones automáticas
